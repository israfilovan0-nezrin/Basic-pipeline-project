# CI/CD Pipeline Project

## CI (Continuous Integration)
Pull request zamanı işləyir:
- kod yoxlanılır
- test və build mərhələləri icra olunur

## CD (Continuous Deployment)
Main branch-ə push edildikdə:
- build edilir
- deployment həyata keçirilir

## Nəticə
GitHub Actions vasitəsilə CI/CD pipeline qurulmuşdur.

This project demonstrates a simple CI/CD pipeline using GitHub Actions.

- CI: Test and build steps
- CD: Deployment using GitHub Pages
